# ArxOS Wallpaper Manager

A native GTK3 wallpaper browser and setter for ArxOS. Browse the set, set your
desktop background with a live preview, run a slideshow, and pull more from
curated sources, all from one small window.

`v0.0.1` · GTK3 · one file, no daemon

## Install

Ships with ArxOS. Standalone on Arch:

```bash
sudo ./install.sh
```

Launch from the menu (**Wallpaper Manager**) or run `arxos-wallpaper`.

## What it does

- Browse and set wallpapers with a live preview grid
- Slideshow with an interval you choose
- **Download more** from curated git sources and Unsplash, with a limit you set so it never floods your disk
